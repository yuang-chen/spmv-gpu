Detailed Description

template<typename vertex_t, typename edge_t, typename weight_t>
struct gunrock::io::matrix_market_t< vertex_t, edge_t, weight_t >

Reads a MARKET graph from an input-stream into a specified sparse format.

Here is an example of the matrix market format +-------------------------------------------—+ |%MatrixMarket matrix coordinate real general | <— header line |% | <–+ |% comments | |– 0 or more comments |% | <–+ | M N L | <— rows, columns, entries | I1 J1 A(I1, J1) | <–+ | I2 J2 A(I2, J2) | | | I3 J3 A(I3, J3) | |– L lines | . . . | | | IL JL A(IL, JL) | <–+ +-------------------------------------------—+

Indices are 1-based i.2. A(1,1) is the first element.
