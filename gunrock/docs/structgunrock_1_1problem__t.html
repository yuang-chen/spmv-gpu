template<typename graph_t>
struct gunrock::problem_t< graph_t >

Problem contains the data structure of the graph algorithms implemented in gunrock. Inherit problem struct for your custom applications' implementation.

Overview
Problem describes the data slice of your aglorithm, and the data can often be replicated or partitioned to multiple instances (for example, in a multi-gpu context).
