template<typename vertex_t , typename edge_t , frontier_kind_t _kind = frontier_kind_t::vertex_frontier, frontier_view_t _view = frontier_view_t::vector>
bool gunrock::frontier::frontier_t< vertex_t, edge_t, _kind, _view >::is_empty ( ) const
inline

Is the frontier empty or not? For certain underlying data structures, this is an expensive operation (for example, boolmap) if the number of elements in a frontier is not self-managed. Meaning, if the user is not keeping track of the number of elements within a frontier.

Returns
true if the frontier is empty.
false if the frontier is not empty.
